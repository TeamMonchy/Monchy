Derby Términos de servicio

  • Última modificación: 21 de abril de 2021.

    • Lo primero que hay que entender es la diferencia entre los Términos y Condiciones y las Políticas de Privacidad de los datos personales.
    El primero se refiere al acuerdo básico entre los usuarios del Bot y el servidor en un ámbito global. la prohibición del uso de imágenes del bot y servidor entre otros. Las Políticas de Privacidad, por su parte, se refieren al tratamiento de los datos de los usuarios que tienen algún tipo de interacción con el bot / servidor: cómo se recaban, cómo se van a almacenar, cuáles van a ser utilizados para y si se van a utilizar. para compartir. Respecto a esto, los datos que podrá obtener el Bot serán: El nombre como, Nombre de usuario, Etiqueta de usuario, (Discriminador) e ID de usuario. El bot no podrá ver ni enviar mensajes en su nombre así como tampoco podrá ver los servidores donde se encuentra el usuario, con esto tendrá la seguridad de usuario al bot como se desee..

    La regla es clara: todos los bots, sea cual sea su desarrollador, deben garantizar el Habeas Data, es decir, el derecho que tienen los usuarios a conocer, actualizar, rectificar y eliminar la información que se haya recabado sobre ellos en bases de datos y archivos; el desarrollador, por su parte, asume el compromiso de guardarlos de forma segura e indicar cuál será el uso que se les dará. Esto indica que se puede solicitar una solicitud para que quienes no quieran o no quieran que el bot o el servidor tengan sus datos puedan solicitarlo al desarrollador, para que los elimine de la base de datos y con ello el usuario pueda tenerlos completos y seguridad confiable del bot y del servidor.



    Responsabilidad por los datos del usuario

    Los usuarios podrán utilizar todos los comandos del bot y en el servidor que deseen, de la misma forma pueden cancelar o habilitar algunos comandos como los comandos de sugerencias de conjuntos, setprefix, logs de canales, etc. para esto el usuario debe proporcionar el información requerida para que El Bot continúe con la ejecución de dicho comando, para esto necesitaría la Información del Servidor como la siguiente: ID del Servidor, Nombre del Servidor Nombre del Autor, ID del Autor, Nombre del Canal, ID del Canal además el Bot puede tener el servidor información como: propietario del servidor, ID de propietario, miembros para esto debe estar de acuerdo con estas condiciones y estar informado para que sea utilizado, de la misma manera si no está de acuerdo con estas condiciones puede consultarlas con el desarrollador / ceo, para que puedas consultar todo lo demás y dudas que surjan.



    El uso de los comandos dados por el Usuario que los ejecuta

    El bot podrá detectar y enviar un mensaje al servidor oficial del mismo que ejecuta un comando del bot, este ya no estará disponible en el servidor oficial del bot donde se encuentra el Desarrollador / Ceo de dicho Bot, para esto el Desarrollador del bot y dichos miembros de la administración de dicho fideicomiso para que puedas tener total privacidad de los comandos que ejecutes con el bot.



    ♡ Mensaje del Desarrollador

    A medida que el Bot y el servidor crezcan cada vez más, las condiciones de uso y términos de privacidad se irán incrementando y mejorando por ahora, estarán las condiciones básicas del bot, cualquier consulta a: Derby Agente de Términos.
